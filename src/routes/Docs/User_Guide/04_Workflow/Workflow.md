## Content States

### Default States
1. **Draft**
   - Initial content state
   - Work in progress
   - Not publicly visible
   - Can be edited freely

2. **Review**
   - Content ready for review
   - Locked for editing
   - Reviewer assigned
   - Comments enabled

3. **Approved**
   - Content verified
   - Ready for publishing
   - Final checks
   - Schedule setting

4. **Published**
   - Live content
   - Publicly visible
   - Version tracked
   - Can be reverted
