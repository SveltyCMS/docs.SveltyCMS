## Content Dashboard

### Accessing Content

1. Navigate to Collections
2. Select your collection
3. View content list

- Content overview

- Quick actions
- Status filters
- Search functionality

### New Entry Creation

1. Click "New Entry" button
2. Choose content type (if applicable)
3. Fill in required fields
4. Save or publish
